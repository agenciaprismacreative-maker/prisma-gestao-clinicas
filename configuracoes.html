Regras da agenda

Senha do gerente

Usada para autorizar ações sensíveis na hora da venda. Como o sistema não tem um servidor próprio, trate essa senha como uma trava operacional para a equipe (evitar desconto/cortesia sem autorização), não como segurança de dados — ela não substitui o login de cada pessoa.

Teto rígido: ninguém consegue aplicar um desconto acima desse percentual em Vendas, nem com a senha do gerente. Deixe em branco para não limitar.

Permissões de visualização

Se desmarcar, só o Administrador enxerga o Desempenho. Cada funcionário sempre vê só os próprios números, nunca os dos colegas.

Some ao critério acima: mesmo com a visibilidade liberada, a pessoa precisa digitar a senha do gerente toda vez que abrir a página. O Administrador nunca precisa dessa senha.
